01. 实战项目概述

实战项目概述

本实战项目的目标是将你编写的二维直方图滤波器 python 代码翻译为 C++ 。

在审阅
Python 代码后，你需要下载一些 C++ 入门代码。你需要完成带有 TODO （待完成）批注的函数，确保直方图滤波器功能正常。

要评估代码的正确性，可以编译 tests.cpp 然后运行输出。具体方法是，在命令行中运行以下代码： 

g++ tests.cpp
./a.out
