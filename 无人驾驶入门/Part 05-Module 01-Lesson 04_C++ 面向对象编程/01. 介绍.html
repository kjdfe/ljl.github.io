01. 介绍

介绍

课程概述

在该课中，你将学习如何用 C ++ 编写类。 本课程假设你已具备了面向对象程序设计的基本知识，如纳米学位早期的“使用矩阵 - 状态和面向对象编程”课程中所教导的知识。

本课的第一部分是自上而下的方法。 我们将向你展示一个 Python 类，然后在 C++ 中比较相同的类。 一旦你了解了 C++ 面向对象的语法，你就可以一步步地编写你自己的 C++ 类。

请记住，无论你使用 Python 还是 C++，面向对象编程的基本原理都是一样的。 在 Python 和 C++ 中，你都可以定义类变量和类方法； 差别主要在语法上。 C++ 拥有一些 Python 不具备的方面，你将在本课中学习到。
