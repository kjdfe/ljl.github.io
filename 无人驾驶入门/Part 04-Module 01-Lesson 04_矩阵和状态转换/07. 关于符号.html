07. 关于符号

关于符号

在之前的视频（以及下一个视频）中，你看到了以下等式：

x' = x + \dot{x}

翻译成大家看得懂的语言，就是：

移动 后 的位置 x ( x' ) 等于移动 前 的位置 ( x ) 加上在 x 方向的速度 ( \dot{x} )。

如果你认真阅读了这个语句，你可能会注意到，它好像不太对，因为它没有考虑到移动的 持续时间 。开 10 秒钟的移动距离肯定比开 1 秒钟的距离更远。

在前面的视频中，我们假设了，移动持续时间 （通常被称作 \Delta t ） 等于 1 秒。上述等式的“完整”版本是：

x' = x + \dot{x} \Delta t

符号 含义
x 移动 前 x 的位置
x' 移动 后 x 的位置
\dot{x} x 方向的速度
\Delta t 移动的持续时间“delta t”
