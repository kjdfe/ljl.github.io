22. 状态向量

状态向量

状态向量

到目前为止，你已经看到一辆汽车的状态包含了我们已经放入 Python 列表中的多个值。

但是，这些值 通常 都包含在一个数据类型中：一个 状态向量 。

向量与 Python 中的列表相似，因为它包含多个值，但它们在数学上差别很大。

什么是向量？

向量是一种矩阵。你可以把它想象成一个数字组成的网格，而矩阵是一种数学数字类型，类似于整数或者浮点数。

和整数或者浮点数类似，矩阵可以相乘、缩放、两两相加、等等。

这叫做 线性代数 ，线性代数将在无人驾驶车系统中不出现。它涉及：

  • 车辆移动运算
  • 深度学习应用，以及
  • 图像和视频分析

在本节中，我们可以看到如何使用状态向量和矩阵来帮助我们有效地预测新状态并定位车辆。
