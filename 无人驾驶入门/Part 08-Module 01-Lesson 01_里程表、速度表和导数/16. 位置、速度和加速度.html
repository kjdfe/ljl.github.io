16. 位置、速度和加速度

位置、速度和加速度

请允许我采用如下 5 种不同表达，讨论 位置 和 速度 。

  1. 速度 是 位移 关于时间的导数。

  2. 速度 是 时间 相关的 位置 变化的瞬时速率。

  3. 一个物体的 速度 能告知我们，它的 位置 随时间会发生多大的变化。

  4. 某个时间的 速度 只是与 位置 vs 时间 的图形相切的直线的斜率。

  5. v(t) = \frac{dx}{dt} = \dot{x}(t)

同样地， 速度 和 加速度 也可以采用如下 5 种不同表达讨论。

  1. 加速度 是 速度 关于时间的导数。

  2. 加速度 是与 时间 相关的 速度 变化的瞬时速率。

  3. 一个物体的 加速度 能告知我们，它的 速度 随时间会发生多大的变化。

  4. 某个时间的 加速度 只是与 速度 vs 时间 的图形相切的直线的斜率。

  5. a(t) = \frac{dv}{dt} = \dot{v}(t)

对于 位置 和 加速度 之间的关系，我们也可以做一些有趣的陈述：

  1. 加速度 是 位置 关于时间的二阶导数。
  2. a(t) = \frac{d^2}{dt^2}x(t) = \frac{d^2x}{dt^2} = \ddot{x}(t)

我们将在下一课中探讨这一点。就目前而言，只要对位置进行两次微分，就可以得出加速度！
